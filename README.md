# Frontend
How to run:

Install dependency
```sh
$ npm install
```
Run app
```sh
$ npm run dev
```
Server run in http://localhost:8080

# Backend
How to run:

Installing virtual environment
```sh
virtualenv -p python3 venv
```
Load environment
```sh
source venv/bin/activate
```
Installing dependency
```sh
pip install -r requirements.txt
```
Run app
```sh
python article.py
```
Server run in http://localhost:5000
